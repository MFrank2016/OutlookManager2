# 用户编辑面板快速指南

## 🎯 快速开始

### 打开编辑面板

1. 访问用户管理页面: `/user-management.html`
2. 点击任意用户的"编辑"按钮
3. 编辑面板自动打开

## ✨ 新功能一览

### 1. 🔍 账户搜索

**位置**: 绑定账户区域顶部

**使用方法**:
```
1. 在搜索框输入关键词（如: "outlook", "gmail"）
2. 列表自动过滤显示匹配的账户
3. 清空搜索框恢复显示所有账户
```

**特点**:
- ⚡ 实时搜索，无需点击按钮
- 🔤 不区分大小写
- 📊 自动更新统计信息

### 2. 📄 分页浏览

**位置**: 账户列表底部

**使用方法**:
```
1. 点击"下一页"查看更多账户
2. 点击"上一页"返回前面的账户
3. 中间显示当前页码（如: 2 / 5）
```

**特点**:
- 📱 每页显示 20 个账户
- 🔄 自动隐藏（总数 ≤ 20 时）
- ✅ 选中状态跨页保持

### 3. 📊 实时统计

**位置**: 搜索框下方

**显示内容**:
```
已选 5 | 显示 50 | 共 100
  ↓       ↓        ↓
选中数  过滤后   总账户数
```

**特点**:
- 🔵 选中数用蓝色高亮
- 🔄 实时更新
- 📈 精确统计

### 4. ⚡ 快捷操作

**位置**: 统计信息下方

**按钮功能**:

#### 全选当前页
```
点击后：选中当前页显示的所有账户
适用场景：快速选择一批账户
```

#### 清空选择
```
点击后：取消所有页面的选择
适用场景：重新开始选择
```

## 🔥 使用技巧

### 技巧 1: 快速绑定多个账户

```
1. 搜索 "outlook.com"
2. 点击"全选当前页"
3. 点击"下一页"
4. 再次点击"全选当前页"
5. 保存
```

**结果**: 所有 Outlook 账户都被绑定

### 技巧 2: 精确选择

```
1. 搜索关键词缩小范围
2. 手动选择需要的账户
3. 清空搜索查看所有账户
4. 选中的账户仍然保持选中
```

**优势**: 避免误选

### 技巧 3: 跨页选择

```
第1页: 选择 5 个账户
第2页: 选择 3 个账户
第3页: 选择 2 个账户
统计显示: 已选 10
```

**说明**: 所有选择都会被保存

### 技巧 4: 批量操作

```
1. 点击"全选当前页" (选中 20 个)
2. 手动取消不需要的 3 个
3. 翻页继续选择
```

**效果**: 高效且精确

## ⚠️ 注意事项

### 1. 搜索不影响选择
```
✅ 正确: 搜索后选择 → 清空搜索 → 选择保持
❌ 错误: 以为清空搜索会清空选择
```

### 2. 翻页不丢失选择
```
✅ 正确: 第1页选择 → 翻页 → 第1页的选择保持
❌ 错误: 以为翻页会丢失之前的选择
```

### 3. 清空选择是全局的
```
✅ 正确: "清空选择"会清空所有页面的选择
❌ 错误: 以为只清空当前页
```

### 4. 统计信息说明
```
已选: 所有页面选中的总数
显示: 当前搜索/筛选后的数量
共: 系统中的账户总数
```

## 🆘 常见问题

### Q1: 搜索后为什么看不到之前选中的账户？

**A**: 如果之前选中的账户不匹配搜索词，它们会被隐藏，但选中状态仍然保持。清空搜索即可看到。

### Q2: 翻页后之前的选择还在吗？

**A**: 在！系统使用全局 Set 存储选中状态，翻页不会丢失任何选择。

### Q3: 如何知道总共选了多少个账户？

**A**: 查看"已选 X"的数字，这是所有页面选中账户的总数。

### Q4: "全选当前页"和"清空选择"的区别？

**A**: 
- "全选当前页": 只选中当前页显示的账户
- "清空选择": 清空所有页面的所有选择

### Q5: 移动端能正常使用吗？

**A**: 完全可以！界面已经过移动端优化，所有功能都支持触摸操作。
